🏞️ 景点详解 · 抚仙湖

抚仙湖是中国最大的深水型淡水湖泊，平均深度90多米，湖水清澈见底，被古人称为"琉璃万顷"。蓄水量相当于12个滇池、6个洱海。水质为国家一类饮用水标准，是中国水质最好的天然湖泊之一。

免费1-1.5h车程老人友好

📌 推荐游玩时间：全天，湖边散步+日落。月亮湾湿地公园免费开放，沙滩细腻，适合全家。
